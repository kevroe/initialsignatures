General Information

This template includes following pages:

  • index.html
  • home-1.html
  • home-2.html
  • home-3.html
  • contact.php

Stylesheets (CSS) of this template are placed in /css/ folder.
All Images are in /img/ folder.
And all Javascripts files are in /js/ folder.

HTML Structure

The template is based on Bootstrap 3 the most popular framework for developing responsive, mobile first projects.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Create a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding, then create .row and add columns with appropriate device name (xs, sm, md, lg) and grid number (2, 4, 6, 12).

For full documentation, please visit Bootstrap 3 site: http://getbootstrap.com/css/ 

<div class="container">
  <div class="row">
    <div class="col-lg-12">
      ...
    </div>
  </div>
</div>

Fullscreen Slider Backgrounds

To change the images for home fullscreen slider go to js/scripts.js line 519.
You can change the links and add more images if you need to, also you can control duration of slider transition. 

$(".home-fullscreen-slider").backstretch([
        "img/slide-1.jpg",
        "img/slide-2.jpg",
        "img/slide-3.jpg",
    ], {
       fade: 750,
       duration: 4000
    });
}

Portfolio

A powerful jQuery plugin that provides beautiful animated filtering, custom captions and it's perfect for portfolios, galleries, team members, blog posts or any other ordered content with dynamic and responsive layouts.

Creating portfolio is quite easy following just two simple steps.
1) Add Portfolio Item to Section
2) Create Portfolio Item Description Page

Adding Portfolio Items to the Section

Find the below code and add it to portfolio list <ul class="grid cs-style-3">

HTML Markup for project item: 

<li class="cbp-item web-desing">
    <figure>
        <img src="img/portfolio/thumb/1.jpg" alt="">
        <figcaption>
            <a href="img/portfolio/full/1.jpg" class="cbp-lightbox" data-title="Our Project 1"><i class="fa fa-search fa-2x"></i></a>
            <a href="projects/project1.html" class="cbp-singlePage"><i class="fa fa-expand fa-2x"></i></a>
        </figcaption>
    </figure>
</li>

Create Portfolio Item Description Page

Each AJAX project description is in projects folder.

Use and edit one of 9 included in projects folder or you can create a new HTML page inside the /projects/ folder.

For loading more projects with "Load More" button check loadMore.html in / projects / folder. Add project items to the <div class="cbp-loadMore-block1"> and <div class="cbp-loadMore-block2">

One notice: Portfolio AJAX projects description page and vimeo - youtube videos will be functional only on your server.
Upload to your server when testing.

Twitter Feed

jQuery Tweetie - Simple Twitter Feed Script that works with new Twitter 1.1 API.

New Twitter API requires oAuth Token Key, so it's three step process.

Step 1

First, you need a consumer key and secret keys. Get one from https://dev.twitter.com/apps

Step 2

In "api" folder edit "config.php" file and replace variables with your Consumer and oAuth Keys.


config.php: 

// Consumer Key
define('CONSUMER_KEY', 'CONSUMER_KEY_HERE');
define('CONSUMER_SECRET', 'CONSUMER_SECRET_HERE');

// User Access Token
define('ACCESS_TOKEN', 'ACCESS_TOKEN_HERE');
define('ACCESS_SECRET', 'ACCESS_SECRET_HERE');
One notice: this will be functional only on your server, php file can not be executed on your local computer.
Upload to your server when testing.
